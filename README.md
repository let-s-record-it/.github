# 📌 Commit Convention

| 깃모지 | 사용예시                 |
| ------ | :----------------------- |
| 🎉     | 프로젝트 시작 |
| ✨     | 새 기능 (feature) |
| 💄     | UI/스타일 파일 추가/수정 (style) |
| ♻️     | 코드 리팩토링 (refactor) |
| 🐛     | 버그 수정 (fix) |
| 🚑     | 긴급 수정 (hotfix) |
| ✅     | 테스트 추가/수정 (test) |
| 📝     | 문서 추가/수정 (docs) |
| 🔧     | 설정 파일 추가/수정 (chore) |
| 🔥     | 코드/파일 삭제 (remove) |
| 🚚     | 코드/파일 이름 변경 (rename) |
